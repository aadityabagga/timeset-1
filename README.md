Its a python-gui for managing system date and time.

#Dependencies-

*gksu

*ntp

*python

*python-gobject

*systemd

#Installation-

Available in the Manjaro Linux repository. 

Available for Arch Linux users in the AUR (https://aur.archlinux.org/packages/timeset-gui/).

#Usage-

(as root)
